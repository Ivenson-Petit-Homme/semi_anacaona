## 💡 TROUBLESHOOTING QUICK TIPS

**Images not showing?**
- Check filenames are EXACT (case-sensitive)
- Make sure they're in `assets/images/` folder
- Try a different image format (PNG instead of JPEG)

**Links not working?**
- Make sure you renamed `index-new.html` to `index.html`
- Check that pages are in the `pages/` folder
- Verify relative paths are correct

**Styles not showing?**
- Refresh browser (Ctrl+F5)
- Clear browser cache
- Make sure `styles.css` is in `assets/css/` folder

**Can't deploy?**
- Try Netlify (easiest - just drag & drop)
- GitHub Pages (free but takes 5 minutes)
- Ask for help if stuck!
